# Get-MtrVideoDeviceInformation

## SYNOPSIS
Get information about all attached Video devices

## SYNTAX

### Set 1
```
Get-MtrVideoDeviceInformation [[-Computer] <String>] [-DeviceID] [<CommonParameters>]
```

## DESCRIPTION
Get information about all attached Video devices

## EXAMPLES

